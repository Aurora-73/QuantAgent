# 突破交易

## 定义
价格突破关键阻力位或前期高点，伴随成交量放大，确认趋势延续的信号。

## 关键要素
- **阻力位**: 前期高点或趋势线上轨
- **成交量确认**: 突破日成交量 >= 5日均量的1.5倍
- **回调验证**: 突破后回踩不破阻力位（支撑转换）

## 入场条件
1. 收盘价站上20日高点
2. 当日成交量 > 20日均量
3. RSI < 70（避免超买区突破）

## 出场条件
1. 止损: 突破位下方 2ATR
2. 止盈: 前高 + 突破幅度的 1:2 风险收益比
3. 移动止损: 20日均线

## A 股适用性
- 适用: 趋势明确的板块（白酒、新能源、AI）
- 慎用: 高度控盘股、ST股
- 假突破率: 约30-40%（需成交量过滤）
