# 海龟交易法则（Turtle Trading）

## 来源
- 作者: Curtis Faith
- 原型: Richard Dennis 的交易培训实验
- 核心: 一套完整的趋势跟踪系统 + 严格的纪律执行

## 核心规则
1. **入场**: 价格突破过去20日高点（系统1）或55日高点（系统2）
2. **止损**: 2ATR 止损
3. **加仓**: 每涨0.5ATR 加仓一次，最多4次
4. **出场**: 跌破过去10日低点（系统1）或20日低点（系统2）

## 仓位管理
- 头寸规模 = (账户的1%) / (ATR × 每点价值)
- 最大风险: 每笔交易不超过总资金的1%

## 借鉴价值
- 机械规则系统可以赚钱（不需要主观判断）
- 一致的仓位管理比入场信号更重要
- 纪律 > 预测
