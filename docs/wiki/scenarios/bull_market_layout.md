# 牛市初期布局

## 识别信号
1. 指数站上年线且年线走平→上翘
2. 成交量持续放大（> 60日均量）
3. 领涨板块出现（券商/科技率先启动）
4. 宏观政策转向宽松

## 布局策略
| 阶段 | 仓位 | 配置方向 |
|------|------|---------|
| 初期确认 | 50% | 券商 + 科技龙头 |
| 趋势强化 | 75% | 加仓主流板块 |
| 加速期 | 90% | 持股不动 |

## 风险管理
- 最大回撤容忍: -8%
- 跌破60日线减仓至50%
- 不追高（避免短期回调）
